## Installation
```bash
$ npm install
```

For development:
```bash
$ npm install --dev
```

**Note:** Maid IRC requires Node.js v0.10 or newer.

## Running
```bash
$ node maid.js
```

**Note:** Please define the node environment variable. Supported variables are `development` and `production`. Will default to production if not defined.
